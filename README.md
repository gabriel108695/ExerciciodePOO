# ExerciciodePOO
1 - O que é um vetor em C++?

e um tipo de dados que e usado para armazenar dados de forma organizado na memoria.

2 - Qual a diferença entre declarar um vetor com valor fixo e inicializá-lo com valores específicos?

a diferença e que se você apenas inicializar o vetor com um valor fixo vai ta informando so o tamanho porem vazio, e inicializando com valores específico você esta guardando dados dentro desse vetor.

3 - Explique o que significa índice e qual o intervalo válido para um vetor int notas[10];

o intervalo válido é de 0 a 9 porque no c++ índice começa do 0 então significa que 0 = 1 posição do vetor.
